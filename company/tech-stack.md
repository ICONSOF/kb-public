# Technology stack

## Core platform

Icons Of is a Microsoft partner and part of the Microsoft global startup programme. The primary development platform is **Microsoft Power Platform** (low-code/no-code) for rapid delivery of sustainability solutions.

Key Microsoft products in use:
- **Microsoft Power Platform** — core development environment
- **Microsoft Cloud for Sustainability** — infrastructure layer
- **Microsoft Sustainability Manager (MSM)** — client-facing sustainability management tool
- **Azure** — cloud infrastructure

## AI and data

- RAG (Retrieval-Augmented Generation) for AI-assisted analysis
- Agent architectures for the MASSIV+ AI guidance layer (AP3)
- AI-augmented data processing (e.g. Scope 3 analysis for Peab)

## MASSIV+ specific

- **Eclipse Data Connector (EDC)** — the IDS connector implementation used for data exchange (same as Catena-X)
- IDS/Gaia-X compliant data space infrastructure
- See `massivplus/architecture/` for technical details

## Partnerships

- **Microsoft** — official partner, startup programme
- **Miljögiraff** — environmental analysis specialists, shared client projects
