# Editing docs — guide for non-git people

You don't need to install anything or know how to use git. Everything can be done directly in the browser on GitHub.

---

## How to edit a file

1. Go to the repository on GitHub
2. Navigate to the file you want to edit
3. Click the **pencil icon** (Edit this file) in the top right corner of the file view
4. Make your changes in the text editor
5. Scroll down to **Commit changes**
6. Write a short description of what you changed (e.g. "Update Ronja persona with new details")
7. Click **Commit changes**

That's it. Your changes are saved and visible to everyone immediately.

---

## How to create a new file

1. Navigate to the folder where the new file should live
2. Click **Add file → Create new file**
3. Give it a name ending in `.md` (e.g. `new-topic.md`)
4. Write your content
5. Commit as above

If you're unsure where a new file belongs, add it to the most relevant existing folder and mention it to someone who can move it if needed.

---

## Text formatting basics

Files use **Markdown** — a simple way to format plain text.

| What you want | What you write |
|---|---|
| Heading | `# Big heading` / `## Smaller heading` |
| Bold | `**bold text**` |
| Bullet list | `- item one` |
| Numbered list | `1. first item` |
| Link | `[link text](https://example.com)` |
| Code or technical terms | `` `inline code` `` |

You don't need to memorise this — GitHub shows a **Preview** tab while editing so you can see how it looks before saving.

---

## Simple rules

- **One topic per file** — if something doesn't fit the file you're editing, it probably belongs in a different file or needs a new one
- **Be specific** — "Updated Ronja persona" is better than "Changes"
- **If something is outdated**, either update it or add a note at the top: `> Note: this was accurate as of [month year], may need updating`
- **Don't delete things** without checking — if you're unsure, add a note rather than removing content
- **Dates** — if you're writing something time-sensitive, include the date so it's clear when it was written
