# Architecture Decision Records (ADRs)

This file logs significant architecture decisions — what we chose, what we considered, and why. Invaluable context for anyone (human or AI) working on the system later.

## Template

```
### ADR-XXX — [Short title]
**Date:** YYYY-MM-DD
**Status:** Proposed / Accepted / Superseded

**Context:** What situation or constraint forced this decision?

**Decision:** What did we decide?

**Alternatives considered:** What else did we look at?

**Consequences:** What are the trade-offs of this choice?
```
