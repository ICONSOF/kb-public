## Key concepts

- **MASSIV+** is a Vinnova-funded initiative (2025–2028) to make Scope 1–2 climate data shareable across supply chains via a federated data space built on IDS/Gaia-X
- **Icons Of** is the technical project lead, responsible for AP2 (data infrastructure), AP3 (AI guidance), and AP4 (pilot)
- **CCDM** (Common Composite Data Model) is the data standard at the heart of MASSIV+
- **IDS Connectors** ("beacons" in MASSIV+) are the technical building blocks for peer-to-peer data exchange
- **SME usability** is a primary design constraint — target is 15-minute onboarding for a small supplier with no IT department
